# 세로형 프로토타입 기본 골격

* 게스트 자동 로그인 및 데이터 로드
* 자동 전투, 일반 몬스터, 5번째 보스
* 사망 후 회복, 스테이지 진행과 골드 보상
* 공격력·체력·공격속도 성장
* 튜토리얼 시작 → 공격 강화 → 적 처치
* 최대 8시간 방치 보상
* 60초 자동 저장
* 전투·성장·가챠·플레이어 허브 UI
* 우편·인벤토리·일일보상·계정 연결
* 기존 가챠 씬 왕복
* 1080×1920 세로 화면 고정
* 중립적인 가챠 플레이스홀더 캐릭터 적용
